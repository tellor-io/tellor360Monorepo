## Overview <a name="overview"> </a>  

This repo imports all Tellor360 contracts for deployment.

## Setting up and deploying

Install Dependencies
```
npm i
```
Compile Smart Contracts
```
npx hardhat compile
```

Deploy contract
```
npx hardhat run scripts/deploy.js --network rinkeby
```
